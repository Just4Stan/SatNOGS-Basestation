## Motor wiring

Wire color mapping (from spec sheet):

| Wire | Function |
|------|----------|
| Red | MOTOR+ |
| Black | MOTOR- |
| Blue | HALL GND |
| Green | HALL VCC |
| White | HALL A (OUT1) |
| Yellow | HALL B (OUT2) |

Encoder output level: **3.3V**.
